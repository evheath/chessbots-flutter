# Chessbots

![screenshot](https://o8jsjsxk9g.ufs.sh/f/Y3NBzhpqodc3wBc0BNWEC53iFNTVDZo7pylu60YrG9Q8qhKm)

A mobile game where users create chessbots that battle each other. Upgrade your bot as you win more battles!

## Tech stack
- Flutter
- Dart
- Firebase
- Firestore
- RxDart
- Crashlytics
- MaterialUI
